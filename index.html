Web Designer Akira Morikawa

東京都在住のWebデザイナー、森川晃です。オンラインスクールでWeb制作について学び、売れるデザインをスピーディーに作成できるように日々特訓中です。ビジュアルデザイン・HTML・CSS・JavaScriptを武器にして豊かなフロントエンドの表現ができるデザイナーを目指しております。

Skills

Web

HTML/CSSでコーディングができます。JavaScript、PHP、WordPressも現在勉強中です。

Design

Illustrator/Photoshopのショートカットキーを駆使し、短時間でかつ直しに強いデザインデータを作ることを意識しております。

Direction

目的となぜこうするのか（したのか）の意図を明確にしデザイナーとクライアントの橋渡し役として尽力してきました。
